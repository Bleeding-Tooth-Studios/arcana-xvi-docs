# Plop Files

Arcana XVI uses Plop to generate files.

## Generators

Arcana XVI has the following generators:

### Component Generator

Generates a new component.

```bash
plop component
```

or

```bash
npx plop component
```

### System Generator

Generates a new system.

```bash
plop system
```

or

```bash
npx plop system
```
